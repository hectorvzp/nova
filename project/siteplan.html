Site Plan: World of Programming Languages

Site Name

Name: World of Programming Languages

This name was chosen because it clearly represents the site's purpose: to present, compare, and teach a wide variety of programming languages. It's accessible for beginners and appealing to more experienced developers.

Optional domain: worldoflanguages.dev

Site Purpose

The purpose of this site is to serve as an educational and informational portal about programming languages. Visitors will be able to explore content such as:

  • History and use of major programming languages
  • Language comparisons
  • Beginner tutorials
  • Recommendations for free courses and resources

Scenarios

  • Which programming language should I learn first?
  • What are the key differences between Python and Java?
  • Where can I find free tutorials for learning C++?

Color Scheme

  • Primary Color: Dark Blue (#2c3e50) – used for titles and highlight buttons.
  • Secondary Color: Light Gray (#f4f4f9) – used as the general background color.

Typography

  • Arial: Used for body text for its clean and readable appearance.
  • Roboto Slab: (to be added via Google Fonts) used for headings, giving a technical and modern look.
